# 🎮 使用指南

**[文档目录](README.md) | [English](../en/usage.md)**

## 创建项目

1. 打开应用首页。
2. 点击 "Create Project"。
3. 输入项目名称和描述。
4. 确认创建。

## 上传视频

1. 进入项目并打开 "Videos" 标签页。
2. 点击 "Upload"。
3. 选择视频文件或文件夹。
4. 等待上传和处理完成。
5. 预览上传的视频列表。

## 标注视频

1. 选择视频并进入标注编辑器。
2. 使用时间线或快捷键浏览帧。
3. 使用点击提示、矩形框、多边形或文本标签创建标注。
4. 配置 SAM 后端后，点击 "Propagate" 将掩码传播到后续帧。
5. 编辑掩码并保存更改。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `A` / `D` | 上一帧 / 下一帧 |
| `Spacebar` | 播放 / 暂停 |
| `Delete` | 删除选中标注 |
| `Ctrl+Z` | 撤销 |
| `Escape` | 取消当前操作 |

## 导出项目

1. 进入项目并打开 "Export" 标签页。
2. 选择导出格式：帧和掩码、JSON 或带叠加层的视频。
3. 点击 "Export"。
4. 下载生成文件。
