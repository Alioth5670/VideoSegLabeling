- 🎥 视频和多媒体项目管理
- 🖱️ 点击、矩形框、文本提示和多边形标注工具
- 🤖 集成 SAM3/SAM3.1 进行 AI 辅助分割

- 📁 项目创建、编辑、删除和导出工作流
- 🔄 批量删除和编辑
- 🖥️ React + TypeScript Web 界面
