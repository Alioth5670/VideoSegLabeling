# 🎮 Usage Guide

**[Documentation](README.md) | [中文](../zh/usage.md)**

## Create a Project

1. Open the application home page.
2. Click "Create Project".
3. Enter a project name and description.
4. Confirm creation.

## Upload Videos

1. Enter a project and open the "Videos" tab.
2. Click "Upload".
3. Select video files or a folder.
4. Wait for upload and processing.
5. Preview the uploaded video list.

## Annotate Videos

1. Select a video and enter the annotation editor.
2. Browse frames with the timeline or keyboard shortcuts.
3. Create annotations with point prompts, boxes, polygons, or text labels.
4. Click "Propagate" to apply masks to subsequent frames when a SAM backend is configured.
5. Edit masks and save changes.

## Keyboard Shortcuts

| Shortcut | Function |
|----------|----------|
| `A` / `D` | Previous / next frame |
| `Spacebar` | Play / pause |
| `Delete` | Delete selected annotation |
| `Ctrl+Z` | Undo |
| `Escape` | Cancel current operation |

## Export Project

1. Enter a project and open the "Export" tab.
2. Select export format: frames and masks, JSON, or video overlay.
3. Click "Export".
4. Download generated files.
