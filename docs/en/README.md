## ✨ Key Features

- 🎥 Video and multimedia project management
- 🖱️ Point, bounding box, text prompt, and polygon annotation tools
- 🤖 SAM3/SAM3.1 integration for AI-assisted segmentation
- 📊 Mask propagation across video frames
- 📁 Project creation, editing, deletion, and export workflows
- 🔄 Batch deletion and editing
- 🖥️ React + TypeScript web interface
